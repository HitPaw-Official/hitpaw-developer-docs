# HitPaw Developer Documentation
 
 This repository contains the official developer documentation for **HitPaw AI Enhancement API**. It provides comprehensive technical guides, API references, and integration examples for developers looking to leverage world-class AI models for image and video processing.
 
 ## Documentation Overview
 
 ### 1. Image Enhancement API
 Dedicated to providing high-fidelity and generative solutions for photographic enhancement, including:
 *   **Super-Resolution:** Intelligent 2x/4x upscaling for general and high-fidelity scenarios.
 *   **Portrait Optimization:** Specialized models for facial recovery, skin texture restoration, and generative face reconstruction.
 *   **Restoration:** Advanced denoising and sharpening techniques for damaged or low-quality photos.
 
 ### 2. Video Enhancement API
 Focused on industrial-grade video restoration with temporal stability, featuring:
 *   **Cinematic Upscaling:** Upscaling SD/HD footage to 4K while maintaining professional-grade texture.
 *   **Temporal Portrait Restoration:** AI-driven facial tracking and enhancement designed specifically for moving subjects.
 *   **General Restoration:** Versatile models for de-noising, de-blurring, and removing complex compression artifacts from video streams.
 
 ### 3. Common Utilities
 Shared services to simplify the integration workflow:
 *   **OSS Pre-sign Upload:** A secure mechanism for clients to upload media resources directly to cloud storage, ensuring high success rates and streamlined data ingestion.
 
 ## Content Maintenance
 
 The project is organized into structured sections:
 *   `/docs/image`: Detailed references for image processing endpoints.
 *   `/docs/video`: Detailed references for video processing endpoints.
 *   `/docs/common`: Documentation for shared infrastructure and auxiliary APIs.
