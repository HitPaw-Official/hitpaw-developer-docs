We have integrated a global Cookie Consent Banner and updated our site footer to improve user privacy control and adhere to security guidelines.

<!-- truncate -->

### Key Changes
- **Cookie Consent Banner**: A sleek, dark glassmorphic banner appears on the first visit to prompt users to accept or reject cookies.
- **Google Consent Mode V2 Integration**: Configured default consent flags to deny ad/analytics storage until the user explicitly accepts the consent banner.
- **Footer Updates**: Added a dedicated **Legal** section containing official links to:
  - [Privacy Policy](https://www.hitpaw.com/privacy-policy.html)
  - [Terms & Conditions](https://www.hitpaw.com/terms-and-conditions.html)
